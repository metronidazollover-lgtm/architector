# Changelog

## 2026-07-15 — этап 3: видеть внутрь

- Превью начинки в теле узла-контейнера (semantic zoom, SVG): дети с именами и фигурами,
  внутренние связи, внуки точкой-индикатором. Порог: зум ≥ 60%.
- Alt+hover peek: начинка узла показывается в полный размер на своих местах,
  соседи приглушаются; отпускание Alt возвращает вид.
- `HierarchyUtils.getChildrenBBox` + тест.

## 2026-07-15 — этап 2: относительные координаты (формат v10)

- Позиции детей хранятся относительно родителя; перенос родителя перемещает всю начинку.
- Миграция v9→v10 на лету: localStorage (`architector_state_v9` → `architector_state_v10`)
  и импорт файлов без `formatVersion`. Мировые позиции при конвертации не меняются.
- Новый экшен `REPARENT_ENTITY`: смена родителя с сохранением мировой позиции и защитой от циклов.
- `MOVE_SELECTED` не двигает потомка выделенного предка дважды; удаление слоя компенсирует
  позиции его детей.
- `HierarchyUtils.getAbsolutePosition` с кэшем на поколение стейта.
- `server.py` + правка `run.bat`: no-cache сервер разработки (браузер кэшировал text/babel-скрипты).
- Экспорт пишет `formatVersion: 10` и `cameraByContext`.

## 2026-07-15 — этап 0: типы и тесты

- Редьюсер вынесен в `store/reducer.js` (без JSX, двойной экспорт браузер/Node).
- JSDoc-типы (`types/`), проверка `npx -y -p typescript tsc -p jsconfig.json`.
- Юнит-тесты `node --test tests/*.test.js`: навигация, камеры, удаления, геометрия, миграция.

## 2026-07-15 — этап 1: быстрые победы

- Бейдж содержимого в шапке узла (счётчик детей и связей), клик — вход внутрь.
- Empty state при входе в пустой узел.
- Esc (каскад: режим → выделение → вверх), Cmd/Ctrl+[ и ] — история посещённых контекстов.
- Камера каждого уровня сохраняется (`cameraByContext`) и восстанавливается при возврате.
