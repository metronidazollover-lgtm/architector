# Карта жестов и хоткеев

Обновляется каждым этапом плана (docs/PLAN.md).

## Мышь

| Жест | Действие |
|---|---|
| Клик по узлу/слою/связи | Выделить |
| Shift+клик | Добавить/убрать из выделения |
| Перетаскивание узла | Переместить (потомки едут вместе, v10) |
| Перетаскивание шапки слоя | Переместить слой (замок: захват соседей по контексту в границах) |
| Уголок справа внизу | Изменить размер |
| Двойной клик по узлу | Войти внутрь (Dive In) |
| Клик по бейджу «⧉ N» в шапке узла | Войти внутрь |
| Двойной клик по порту/связи | Войти в контекст порта/связи |
| Перетаскивание от порта | Создать связь (умное создание порта при сбросе на узел) |
| Колесо | Зум в точку курсора |
| Средняя кнопка / Shift+ЛКМ по холсту | Панорама |
| Клик по пустому холсту | Сброс выделения |

## Клавиатура

| Клавиши | Действие |
|---|---|
| Esc | Каскад: выйти из режима → снять выделение → на уровень вверх |
| Cmd/Ctrl+[ | Назад по истории посещённых контекстов |
| Cmd/Ctrl+] | Вперёд по истории |
| Cmd/Ctrl+C / V | Копировать / вставить узел |
| Cmd/Ctrl+Z | Отмена |
| Cmd/Ctrl+Y, Cmd/Ctrl+Shift+Z | Повтор |
| Delete / Backspace | Удалить выделенное |

## Навигация по уровням

- Хлебные крошки сверху: клик — переход на уровень, глаз — X-Ray просвечивание уровня.
- Камера каждого уровня запоминается и восстанавливается при возврате (v10).
- Вход в пустой узел показывает подсказку с кнопками «Открыть библиотеку» и «Наверх».

## Тач

- Один палец по холсту — панорама, два пальца — зум (pinch).
