Основной стек нашего приложения (Architector) следующий:

React 18 (работает без сборки, подключается напрямую через CDN)
TailwindCSS (для стилизации интерфейса)
Lucide Icons (для иконок)
React Context + useReducer (пользовательская реализация паттерна Flux/Redux для управления состоянием приложения)
Все это работает полностью в браузере без использования дополнительных инструментов сборки (Webpack, Vite и т.д.) или бэкенда.
