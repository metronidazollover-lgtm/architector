Конечно! Вот подробная документация по терминологии и элементам нашего приложения, которую вы можете использовать для составления точных промптов:

🧩 Основные визуальные элементы (Сущности)
Холст (Canvas) — бесконечное рабочее пространство, на котором располагаются все элементы. Имеет координаты смещения (offset) и масштаб (zoom).
Слой (Layer) — визуальная группа или фрейм. Представляет собой большую подложку, поверх которой располагаются узлы, что помогает структурировать проект.
Узел (Node) — основной прямоугольный блок на холсте (например, микросервис, конфигурация или база данных). 
Порт (Port) — точка подключения на гранях узла (сверху, снизу, слева, справа). Бывает двух типов: Вход (input) и Выход (output).
Связь (Link) — линия, соединяющая два порта.
ИИ-Ассистент (AI Agent / Copilot) — особый тип узла (type: 'ai-agent'), который содержит внутри себя интерфейс чата и настройки API для общения с нейросетью.
🗂 Интерфейс и Панели
Панель свойств (Property Panel) — выезжающая панель справа. Появляется при выделении узла, порта или связи для редактирования их атрибутов.
Обозреватель / Библиотека (Library) — выезжающая панель слева. Содержит две вкладки: "Объекты" (дерево всех узлов, портов и связей) и "История" (лог действий).
Панель инструментов (Toolbar) — вертикальное меню слева (кнопки создания узла, порта, ИИ-узла, импорта/экспорта и сброса).
Хлебные крошки (Breadcrumbs) — навигационная панель сверху (например, Главный холст > Название узла), показывающая текущий уровень вложенности (контекст).
⚙️ Свойства элементов (Атрибуты)
Для Слоя:

ID (например layer-123)
Name (Название слоя)
Content (Текстовое описание слоя)
Color (Основной цвет, применяемый с полупрозрачностью к фону)

Для Узла:

ID (системный идентификатор, например node-12345)
Name (Название узла)
Group (Группа/Папка — для сортировки в Обозревателе)
Content (Содержание/Текст — внутреннее текстовое описание узла)
Media URL (Ссылка на изображение, отображаемое внутри узла)
Color (Цвет фона узла)
Для Порта:

ID
Name (Название порта)
Content (Описание порта)
Type (Тип: input или output)
Edge (Грань узла, на которой находится порт: top, bottom, left, right)
Position (Позиция на грани в процентах от 0 до 1)
Для Связи:

ID
Name (Название линии)
Content (Описание передаваемых данных)
Color (Цвет линии)
🕹 Действия и Механики (Экшены)
Погружение (Dive Into) — двойной клик по узлу, порту или связи. Переносит пользователя "внутрь" этого элемента (смена контекста), скрывая остальные элементы холста, не относящиеся к этому контексту.
Смена контекста (Change Context / Go To Context) — перемещение между уровнями вложенности (через "хлебные крошки" или двойной клик). Узел верхнего уровня называется root (Главный холст).
Панорамирование (Pan) — перемещение камеры по холсту (с помощью зажатого колесика мыши, Shift+ЛКМ или свайпа одним пальцем на тачскрине).
Масштабирование (Zoom) — приближение/отдаление (колесиком мыши или щипком двумя пальцами).
Выделение (Select) — клик по элементу, делает его активным и открывает его свойства в Панели свойств.
Центрирование (Center on entity) — автоматическое перемещение камеры к выбранному элементу (происходит при клике на элемент в Обозревателе).
Скрытие (Hide/Unhide) — визуальное отключение узла на холсте через иконку "глаза" в Обозревателе.
Копирование / Вставка (Copy / Paste) — дублирование узлов вместе с их визуальными свойствами (Ctrl+C / Ctrl+V).
Вы можете использовать эти термины в своих промптах (например: "Сделай так, чтобы при выделении Связи ее цвет менялся на красный" или "Добавь новое поле 'Теги' в Панель свойств Узла").
